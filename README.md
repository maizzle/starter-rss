## About

Starter project for [creating an HTML email newsletter from an RSS feed](https://maizzle.com/guides/rss-feed) with Maizzle.

## Getting Started

Run this command:

```bash
npx create-maizzle
```

## Development

Start a local development server:

```
npm run dev
```

Build for production:

```
npm run build
```
